# 发布日志

## v0.1.5 (2026-04-12)

### 新增功能
- ✨ MCP页面嗅探实时日志展示
  - 右侧Drawer面板展示MCP分析全过程日志
  - WebSocket实时推送分析进度和结果
  - 自动提取页面-组件双向关联信息
  - 浮动按钮控制日志面板展开/收起

### 优化改进
-  MCP分析流程优化
  - WebSocket连接Promise化，确保连接建立后再调用API
  - 异步日志回调机制，支持await异步函数
  - 去除Modal确认框，简化交互流程
  - 日志面板自动滚动到底部

- 🎨 UI交互优化
  - 日志面板从底部改为右侧展开（500px宽度）
  - 解决Drawer与Modal的z-index冲突问题
  - 优化loading状态管理，避免页面卡死
  - 添加调试日志，方便问题排查

- 🔧 架构改进
  - MCPPageExplorer构造函数支持可选log_callback参数
  - WebSocket广播增加频道存在性检查
  - 组件关联数据从WebSocket日志中实时提取
  - 双向关联映射（页面→组件、组件→页面）

### 问题修复
- 🐛 修复WebSocket频道不存在导致日志不显示
- 🐛 修复API调用时序问题（先调用API后连接WebSocket）
- 🐛 修复日志面板关闭后无法重新打开
- 🐛 修复MCP分析结果不显示组件关联关系
- 🐛 修复页面刷新后项目信息加载卡死

### 技术细节
- 前端：Vue3 + TypeScript + Ant Design Vue
- 后端：FastAPI + WebSocket实时推送
- 通信：WebSocket频道管理（mcp_{project_id}）
- 日志：5级日志系统（info/success/warning/error/debug）
